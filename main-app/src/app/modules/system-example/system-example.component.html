Технологический процесс обслуживания судна включает ряд событий:
  • 1. Погрузочно-разгрузочные работы.
  • 2. Дополнительные операции у причала (крепление груза, перешвартовка и др.).
  • 3. Необходимый интервал времени между подходами судов.
Расчет вероятности обслуживания судов:
Для расчета вероятности использованы следующие обозначения:
  • λ— интенсивность входящего потока заявок (судов);
  • μ— интенсивность обслуживания заявок (судов);
  • t — время обслуживания;
  • — среднее время пребывания заявки в очереди (при наличии очереди);
  • — вероятность того, что все каналы свободны;
  • ρ — приведенная интенсивность, нагрузка СМО;
  • — вероятность обслуживания;
  • n — количество каналов (причалов);
  • m — длина очереди;
  • — среднее число заявок в очереди.
